Welcome to Flask-Websockets
===========================

Welcome to Flask-Websockets' documentation.

Flask-Websockets is a Flask_ extension_ to easily integrate WebSockets_ using gevent-websockets_.

Get started with :doc:`installation` and then get an overview with the :doc:`quickstart`.
The rest of the docs describe Flask-Websockets' :doc:`api`.

.. _Flask: https://flask.palletsprojects.com
.. _extension: https://flask.palletsprojects.com/en/1.1.x/extensiondev/
.. _WebSockets: https://developer.mozilla.org/en-US/docs/Web/API/WebSockets_API
.. _gevent-websockets: https://gitlab.com/noppo/gevent-websocket
